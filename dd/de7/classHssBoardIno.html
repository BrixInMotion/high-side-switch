HssBoardIno::HssBoardIno ( BtsVariants_t *  variantSel)

Default constructor of the Arduino High-Side-Switch-Board.

This default constructor defines all necessary things for the Profet-Shield. He is meant for all controllers that support the Arduino-Uno form factor. But it is still necessary to chose the variant of the High-Side-Switch.
